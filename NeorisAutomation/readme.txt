# Neoris

#Se realiza la entrega de dos Features en los cuales lee el archivo Excel con los siguientes parámetros

Usuario: usuario que inicia sesión

password: password que inicia sesión

Product: primer producto que busca para agregar al carrito

Product 2: segundo producto que busca para agregar al carrito

Nombre: nombre de la persona que realizará la compra

Apellido: Apellido de la persona que realizará la compra

zipcode: zip code de la persona que realizará la compra


#Todas estas operaciones se programan desde el excel, el cual, leerá de manera dinámica los valores propuestos en dicho excel

Para ejecutar se debe ejecutar el runner con nombre AddPurchaseRunner que se encuentra ubicado en src/test/java/com/co/neoris/runners
